# 萬用爬蟲 - Mac 桌面版

可在 Mac 上雙擊執行的獨立應用程式，可**選擇儲存位置**並將結果直接寫入該資料夾。

## 使用方式

1. **從專案執行（開發）**
   - 在專案根目錄已安裝依賴（含 `cheerio`）的前提下，進入本目錄並安裝 Electron：
     ```bash
     cd electron-app
     npm install
     npm start
     ```
   - 視窗開啟後：輸入起始網址、選擇器、最多頁數與格式，按「選擇資料夾」選儲存位置，再按「開始爬取」。

2. **打包成 Mac .app**
   - 在 `electron-app` 目錄執行：
     ```bash
     npm run dist
     ```
   - 產出在 `dist/` 目錄（例如 `萬用爬蟲-1.0.0.dmg` 或 `萬用爬蟲.app`）。
   - 安裝後可從「應用程式」或 DMG 內拖到應用程式資料夾執行。

## 與網頁版的差異

| 項目       | 網頁版               | Mac 桌面版           |
|------------|----------------------|----------------------|
| 儲存位置   | 瀏覽器下載（由使用者另存） | 可選任意資料夾，直接寫入 |
| 單次頁數   | 最多 50 頁           | 最多 500 頁（可調）   |
| 執行環境   | 伺服器               | 本機                 |

其餘條件（要／不要的 URL、欄位選擇器等）與 CLI / 網頁版設定概念相同。
