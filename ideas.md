# 台中建案導覽與比較平台 - 設計構思

## 專案背景
- 業主：陳學韜，永慶不動產西屯未來店，五義地產
- 目標：建立一個專業的台中市建案導覽與比較平台
- 資料量：437 個建案，涵蓋台中市各行政區與重劃區
- 核心功能：建案瀏覽、篩選、詳情、比較、Line 聯絡

## 方案一：「溫潤信賴」— 日式無印風格

### Design Movement
受日本無印良品（MUJI）與北歐極簡主義啟發，強調「去裝飾化」的溫暖質感。以大量留白、自然色調與精緻的排版層次，傳遞「值得信賴的專業顧問」形象。

### Core Principles
1. **溫暖的中性色調**：以奶油白、暖灰、淺木色為主調，避免冰冷的純白或深黑
2. **呼吸感留白**：每個元素之間保持充足的間距，讓資訊自然呈現而非擁擠堆疊
3. **精緻的文字層次**：透過字重、字距、行高的微妙變化建立視覺層級
4. **觸感質地**：使用細膩的紙張紋理、淡淡的陰影，營造「觸手可及」的溫度

### Color Philosophy
- 主色：奶油白 `#FAF7F2` — 溫暖而不刺眼的背景
- 強調色：永慶黃 `#F5A623` — 品牌識別，用於 CTA 按鈕與重點標記
- 文字色：炭灰 `#2D2D2D` — 沉穩可讀
- 輔助色：暖灰 `#9B9B9B`、淺木色 `#D4C5A9`

### Layout Paradigm
- 首頁採用「雜誌式」垂直滾動佈局，大圖搭配文字區塊交錯排列
- 列表頁使用「瀑布流卡片」，每張卡片高度不等，營造自然的瀏覽節奏
- 詳情頁採用左右分欄：左側固定資訊面板，右側滾動內容
- 比較頁使用水平捲動的卡片並排

### Signature Elements
1. **圓角柔和卡片**：所有卡片使用 16px 圓角，搭配極淡的投影
2. **細線分隔符**：使用 1px 的暖灰細線取代粗重的分隔
3. **手寫風標記**：重點標語使用手寫風底線裝飾

### Interaction Philosophy
- 所有互動都是「輕柔」的：hover 時卡片微微上浮 2px
- 篩選器使用滑順的展開/收合動畫
- 頁面切換使用淡入淡出，不使用劇烈的滑動

### Animation
- 進場動畫：元素從下方 20px 處淡入，持續 400ms
- 卡片 hover：translateY(-2px) + 陰影加深，200ms ease
- 按鈕 hover：背景色漸變 150ms
- 頁面轉場：opacity 0→1，300ms

### Typography System
- 標題字體：Noto Serif TC（思源宋體）— 傳遞文化質感與信賴
- 內文字體：Noto Sans TC（思源黑體）— 清晰易讀
- 數字字體：DM Sans — 現代感的數據呈現
- 標題字重：700，內文字重：400，標籤字重：500

## 方案二：「城市脈動」— 都會建築風格

### Design Movement
受當代建築雜誌（如 Dezeen、ArchDaily）與高端房地產品牌啟發。以深色背景搭配金屬質感的強調色，營造「高端地產顧問」的專業形象。大膽的排版與幾何構圖，讓每個建案都像一件建築作品般被展示。

### Core Principles
1. **暗色沉穩基底**：深灰至近黑的背景，讓建案圖片與數據成為視覺焦點
2. **金屬質感強調**：使用金色或銅色漸層作為強調色，傳遞豪宅級質感
3. **幾何構圖**：大膽的網格線、斜切區塊、不對稱佈局
4. **數據視覺化**：將建案數據（坪數、戶數、樓高）以圖表或大字呈現

### Color Philosophy
- 主色：深炭 `#1A1A1A` — 沉穩的背景
- 強調色：香檳金 `#C9A96E` — 豪宅級質感
- 文字色：象牙白 `#F5F0EB` — 暗底上的清晰文字
- 輔助色：中灰 `#4A4A4A`、暗金 `#8B7355`

### Layout Paradigm
- 首頁採用全螢幕「分屏式」佈局，左側大圖右側文字
- 列表頁使用嚴謹的 3 欄網格，卡片統一高度
- 詳情頁採用全寬圖片 + 下方資訊區塊的「建築作品集」風格
- 比較頁使用深色表格，數據以大字體呈現

### Signature Elements
1. **斜切分隔線**：區塊之間使用 5° 傾斜的分隔，營造動態感
2. **金色細線裝飾**：標題下方使用 2px 金色底線
3. **大數字展示**：關鍵數據（如 437 個建案）以超大字體展示

### Interaction Philosophy
- 互動帶有「力量感」：hover 時卡片放大 1.02 倍
- 篩選器使用滑動抽屜從側邊推入
- 滾動時觸發視差效果，背景圖片以不同速度移動

### Animation
- 進場動畫：元素從左側 30px 滑入 + 淡入，500ms cubic-bezier
- 卡片 hover：scale(1.02) + 金色邊框出現，250ms
- 數字計數器：從 0 滾動到目標數字，800ms
- 區塊分隔：視差滾動效果

### Typography System
- 標題字體：Playfair Display — 優雅的襯線體，傳遞高端感
- 中文標題：Noto Serif TC — 搭配 Playfair 的東方質感
- 內文字體：Noto Sans TC — 暗底上的清晰閱讀
- 數字字體：Oswald — 強而有力的數據展示
- 標題字重：800，內文字重：300，數據字重：600

## 方案三：「暖陽台中」— 在地溫暖品牌風格

### Design Movement
融合永慶不動產的品牌識別與台中在地的溫暖氛圍。以永慶的經典黃色為核心，搭配溫暖的奶油色系與現代簡約的排版，打造一個「親切專業、在地深耕」的品牌形象。不追求豪華感，而是強調「值得信賴的鄰居」。

### Core Principles
1. **品牌一致性**：以永慶黃為核心色彩，讓客戶一眼認出品牌歸屬
2. **溫暖親和力**：奶油色背景搭配圓潤的元素，傳遞「好鄰居」的親切感
3. **資訊優先**：所有設計決策都以「消費者能否快速找到資訊」為最高準則
4. **行動導向**：每個頁面都有明確的 CTA（加入比較、預約看屋、Line 聯絡）

### Color Philosophy
- 主色：永慶黃 `#FFB800` — 品牌核心，用於導航列、CTA、重點標記
- 背景色：暖奶油 `#FBF8F1` — 溫暖不刺眼的閱讀背景
- 卡片色：純白 `#FFFFFF` — 卡片底色，與背景形成微妙層次
- 文字色：深棕灰 `#3D3529` — 溫暖的深色文字
- 輔助色：淺棕 `#E8DFD0`、中棕 `#A69580`
- 成功色：橄欖綠 `#6B8E5A` — 用於「已加入比較」等正向反饋

### Layout Paradigm
- 首頁採用「故事式滾動」：英雄區（開幕照）→ 熱區快選 → 精選建案 → 關於學韜
- 列表頁使用「響應式卡片網格」：桌面 3 欄、平板 2 欄、手機 1 欄
- 詳情頁採用「標籤頁式」：基本資訊 / 五大標語 / 公設學區 / 位置交通
- 比較頁使用「固定首欄」的橫向表格，建案名稱固定在左側

### Signature Elements
1. **黃色標記線**：重要文字下方使用半透明黃色螢光筆效果
2. **圓角友善卡片**：12px 圓角 + 淡淡的暖色投影，營造親和感
3. **品牌角標**：每張建案卡片右上角帶有永慶黃的小標籤（如「14期」、「新成屋」）

### Interaction Philosophy
- 互動是「友善回應」的：hover 時卡片邊框變為永慶黃
- 「加入比較」按鈕點擊後有彈跳動畫，給予正向反饋
- 底部比較欄（Dock）固定在螢幕下方，已選建案以縮圖呈現
- Line 聯絡按鈕始終浮動在右下角

### Animation
- 進場動畫：元素從下方 15px 淡入，350ms ease-out
- 卡片 hover：border-color 變為 #FFB800，box-shadow 加深，200ms
- 比較欄展開：從底部滑入，300ms spring 動畫
- 標籤切換：內容淡入淡出，200ms
- 數字統計：從 0 計數到目標值，600ms

### Typography System
- 標題字體：Noto Sans TC Bold — 清晰有力的中文標題
- 內文字體：Noto Sans TC Regular — 舒適的閱讀體驗
- 英文/數字字體：DM Sans — 現代感的數據與英文
- 標題字重：700，副標題：600，內文：400，標籤：500
- 行高：標題 1.3，內文 1.7（中文需要更大行高）

## 選擇結果

**選擇方案三：「暖陽台中」— 在地溫暖品牌風格**

理由：
1. 與永慶不動產的品牌識別高度一致，客戶一眼就能認出品牌歸屬
2. 溫暖的奶油色系符合用戶偏好的「溫暖的奶油色」設計風格
3. 資訊優先的設計哲學最適合 437 個建案的大量資料展示
4. 行動導向的設計能有效將瀏覽者轉化為諮詢客戶
5. 親切而非高冷的風格，更符合永慶「先誠實再成交」的品牌精神
